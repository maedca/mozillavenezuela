This plugin will give your blog 100% IRC Functionality through the Mibbit AJAX IRC Client

== Description ==

No Java, no flash. With Mibbit, chatting online is easy, just choose your nickname and press go! No sign-up is required and you can use it from wherever you are with just a web browser.

Mibbit has many customizable features which can help you make the most of your chat. You can change skins and use real-time language translation, image thumbnails, or even embed Youtube clips and exchange files with other Mibbit users. 

I Turned the mibbit widget into a fully featured wordpress plugin. Mibbit is an AJAX IRC Client for use in a web browser. The End-User does not need to have anything installed, No JAVA. 

Mibbit is a very fast and durable client and Will provide your users a way to chat in real time.

== Installation ==

1. Upload `wp_mibbit` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<!-- irc_chat -->` on any page
